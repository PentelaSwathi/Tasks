HTML:
HyperText Markup Language, is the standard language used to create and design web pages. It consists of a series of elements (tags) that define the structure and content of a web page.
Heading Tags:
Heading tags in HTML are used to define headings and subheadings on a web page.
  1. h1: 2em (approximately 32px)
  2. h2: 1.5em (approximately 24px)
  3. h3: 1.17em (approximately 18.72px)
  4. h4: 1em (approximately 16px)
  5. h5: 0.83em (approximately 13.28px)
  6. h6: 0.67em (approximately 10.72px)
Paragraph tag:

The tag helps to break up text into manageable chunks, making it easier to read and understand. Each paragraph typically covers a single idea or topic.

Formatting tags:
These tags help structure the content of your web page, making it more readable and visually appealing.
  1. Bold tag:Defines bold text. It doesn't convey any extra importance.
  2. Italic tag:Defines italic text. It doesn't convey any extra emphasis.
  3. Underline tag:Defines underlined text.
  4. Delete tag:Defines text that has been deleted from a document. Browsers typically render this text with a strikethrough.
  5. Quotation tag:Defines a short inline quotation. Browsers typically enclose the text in quotation marks.
  6. Preformatted text tag:Defines preformatted text. Text within this element is displayed in a fixed-width font and preserves both spaces and line breaks.
  7. Blockquote tag: Defines a section that is quoted from another source. Browsers typically indent this text.
  8. Code tag:Defines a piece of computer code. Browsers typically render this text in a monospace font.
  9. Keyboard input tag:Defines keyboard input. Browsers typically render this text in a monospace font.
  10. Mark tag:Defines text that should be marked or highlighted.
Types of tags:
  • Void tags:Void tags, also known as self-closing tags, do not have a closing tag and cannot contain any content.
    • Ex:img,br,hr tags
  • Paired tags:Paired tags, also known as container tags, have both an opening tag and a closing tag. They can enclose content and other elements.
    • Ex:div,p,a tags
  • Block tags:Block tags create a block-level element, which means they start on a new line and take up the full width available (by default), pushing any following content down to a new line.
  • Inline tags:Inline tags create an inline element, which means they do not start on a new line and only take up as much width as necessary.
Tables
It is used to define data in table format.
Uses:
  • Form Layouts
  • Display of Multidimensional Data
Attributes of tables:
  1. Border: Specifies the width of the border around the table cells.
  2. Cellpadding: Specifies the space between the cell content and its borders.
  3. Cellspacing: Specifies the space between cells.
  4. Height: Specifies the height of the table.
  5. Width: Specifies the width of the table.
  6. Align: Specifies the alignment of the table on the page (deprecated in HTML5).

Anchor tag:
It is used to navigate from one page to another page with the help of links.
Attributes of anchor tag:
    href (Hypertext Reference):
  1. Specifies the URL of the page the link goes to.
  2. Can be an absolute URL (e.g., https://www.example.com) or a relative URL (e.g., page.html).
    target:
  1. _self: Opens the link in the same frame (default).
  2. _blank: Opens the link in a new tab or window.
  3. _parent: Opens the link in the parent frame.
  4. _top: Opens the link in the full body of the window.
Image tag:
An image tag is an HTML element used to embed images in web pages.
Attributes of image tag:
  1. src: Specifies the path to the image file.
  2. alt: Provides alternative text for the image if it cannot be displayed.
  3. width: Sets the width of the image in pixels or as a percentage.
  4. height: Sets the height of the image in pixels or as a percentage.
  5. title: Adds a tooltip that appears when the user hovers over the image.
