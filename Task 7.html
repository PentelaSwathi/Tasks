Vs Code:

Visual Studio Code (VS Code) is a popular code editor with numerous features that enhance productivity and make coding more efficient.

Features of Vs Code:

  1. User Interface
  2. Editing Features
  3. Debugging
  4. Extensions
  5. Customization
  6. Workspace Management

Shortcut keys

Keys Description
Ctrl +C For Copying
Ctrl +N For opening New Window
Windows +E Open a File Explorer
Ctrl + Shift + N Create new folder
Alt + F4 Close the active item or exit the active app
Ctrl + P Print
Ctrl +V For Pasting
Ctrl +X For Cutting
Ctrl +A For Selecting All
Ctrl +S For Saving
